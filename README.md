# koa-http2

### 改造koa

```code
// 重写Koa的listen方法
class KoaOnhttp2 extends Koa {
  constructor() {
    super()
  }

  listen(...args) {
    debug('listen');

    const server = http2.createServer(
      keysOptions,
      this.callback()
    );
    return server.listen(...args);
  }
}

const app = new KoaOnhttp2();
```
